# Flexibility routine web app

This is the first React project i created: a minimalsitic website, that fetches yoga stretches from a server and walks you through them in a guided flexibility routine.
It was a great beginner project to learn the barebones of front-end development.

### Flexibility routine

The guided flexibility routine. The user can either follow the given flow of the routine, or click on one of the Stretches in the list to jump to a specific stretch. You can also click on the preview for the next pose, to skip the current pose.
